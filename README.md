# SQL para QA Portfolio 🗄️

Este repositorio recopila ejercicios y consultas SQL diseñadas para la validación de bases de datos e integridad de la información, habilidades fundamentales para el rol de QA Técnico.

## 📁 Contenido del Proyecto
* **[Ejercicios de Consultas SQL (PDF)](./consultas_ejemplo.sql.pdf)**: Documentación de resolución de desafíos técnicos de bases de datos, incluyendo filtrado de datos, uso de condicionales y manejo de tablas.

### 🔍 Habilidades Demostradas:
- **Consultas DQL:** Uso de `SELECT`, `WHERE`, `ORDER BY`.
- **Validación de Datos:** Filtrado de registros específicos según criterios de negocio.
- **Resolución de Problemas:** Aplicación de lógica SQL para extraer información precisa de modelos relacionales.

---
*Herramientas utilizadas: SQL Server, HackerRank para validación de lógica.*
